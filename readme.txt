Sitio web creado para una marca de desarrolladores, se utilizaron las siguientes tecnologias:
-html
-css
-bootstrap
-scss

La pagina de "Elitsum" consiste en una sitio conformado por 5 paginas, entre las cuales se encuentran: 
-Home : un resumen rapido que busca impactar al cliente atrapandolo y darle una via rapida para las siguientes 
paginas y que pueda conocernos hasta contactarnos directamente

-Nosotros : Una pagina dedicada a la informacion de la empresa.

-Portfolio : en Portfolio se encuentran los trabajos realizados por la empresa.

-Servicios : Una pagina con todos los servicios que se ofrecen.

- Contacto : Un formulario presentado de forma comoda para que el cliente rapidamente se pueda poner en contacto con la empresa.


Proyecto creado para la entrega final del curso de desarrollo web de Coderhouse
